TECHNOLOGY STACK:
-----------------
- PHP 8.0+
- MySQL 5.7+
- Apache with mod_rewrite
- JWT Authentication
- PDO for database access

DATABASE TABLES:
----------------
1. users         - User accounts, balances, VIP tiers
2. products      - Products for brand identification tasks
3. tasks         - Completed annotation tasks
4. vip_tiers     - VIP membership tier definitions
5. transactions  - Earnings and withdrawal records

API ENDPOINTS:
--------------

PUBLIC:
POST   /api/auth/register          Register new user
POST   /api/auth/login             User login
GET    /api/products               Get active products
GET    /api/vip-tiers              Get VIP tiers

AUTHENTICATED (requires JWT token):
GET    /api/auth/profile           Get user profile
POST   /api/tasks/submit           Submit completed task
GET    /api/tasks                  Get user's tasks

ADMIN (requires admin token):
GET    /api/admin/stats            Platform statistics
GET    /api/admin/users            List all users
PUT    /api/admin/users/:id        Update user
DELETE /api/admin/users/:id        Delete user
GET    /api/admin/products         List products
POST   /api/admin/products         Create product
PUT    /api/admin/products/:id     Update product
DELETE /api/admin/products/:id     Delete product
GET    /api/admin/tasks            List all tasks
